INSTRUCCIONES PARA LAS IMÁGENES DE PRODUCTOS
==========================================

Para que la tienda de suplementos funcione correctamente, necesitas colocar aquí las imágenes de los siguientes productos:

- stamina_small.png (Barrita de stamina pequeña)
- stamina_large.png (Barrita de stamina grande)
- disquette_oblak.png (Disquete de Oblak para porteros)
- disquette_puyol.png (Disquete de Puyol para defensas)
- disquette_iniesta.png (Disquete de Iniesta para centrocampistas)
- disquette_falcao.png (Disquete de Falcao para delanteros)

Recomendaciones para las imágenes:
- Formato PNG recomendado
- Tamaño 128x128 píxeles o similar
- Diseño colorido que sea fácil de distinguir

Si no tienes las imágenes, el juego seguirá funcionando pero aparecerán espacios vacíos en las tarjetas de productos.

La tienda usa como fondo la imagen "tienda.png" que debe estar en backgrounds/
